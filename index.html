Response Curve Graphing

Here you can see response curve graphs for mapping normalized (0-1) inputs to their marginal utility score (0-1) outputs. All curves have slope arguments, some have intercept arguments. Can use the min, max, and input fields to find the utility value for a particular game state value given min and max bookends for that value.
For example, if we wanted to check out how a response curve might evaluate a health consideration (like an action to heal):
min: 0 (minimum hp is 0)
max: 100 (max hp is 100)
curve: quadratic, slope 1, y intercept 0, x intercept 1 Then, if we have low hp, like 5, we'll have a high score (.9025) for this action.
But if we have high hp, like 90, we'll have a low score (.009) for this action.
